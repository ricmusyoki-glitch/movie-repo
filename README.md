# Ric's Movie hub
This is a movie discovery website which is meant to help users easily find something good to watch.
Nowadays there is difficulty in choosing what to watch because it can be confusing.
Ric's movie hub solves the difficulty by displaying the latest trending movies,The most popular ones and different movie categories in one website.

## Features
View trending movies.
Explore movie categories such as Animation and Action movies.
Movie posters that are fetched from an API
"Watch trailer" button which directs you to youtube to watch the trailer.
Responsive and user-friendly layout

## Technologies used
HTML.
CSS.
Javascript.
Movie API (TMDB).
Youtube for trailer videos.

## Purpose of the project
The purpose of Ric's movie hub is to make movie selection easier for users by organizing movies into
 categories and highlightingwhat is trending and popular.
Instead of wasting time searching across multiple platforms userscan quickly discover movies in one place.

## How to use it
Open the website in your browser.
See the trending and top rated movies.
Visit the movie page for categories.
Click the "Watch trailer" button to see the trailer of the movie you want.
